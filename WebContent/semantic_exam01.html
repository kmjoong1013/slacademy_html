JAVA 관련 교육 내용

      JAVA

JAVA를 이용해서 프로그래밍의 문법과 메소드 단위의 로직을 처리하는 방법을 배우고 객체지향의 개념과 객체단위로 로직을 처리하는 방법을 배운다.

      HTML5

다양한 기기가 활성화 되면서 HTML의 표준화의 필요성이 제기되 었으며 기존 HTML4의 표준화의 요청에 의해 HTML5가 웹표준으로 권장되기 시작하면서 많은 이슈가 되고있다.

      CSS3

HTML5로는 내용과 구성을 만들고 레이아웃의 위치나 배치 디자인등은 css를 활용하는것이 효율적이며 HTML5와 같이 제시 되고있는것이 CSS3이다.

      ORACLE

IT분야에서 DBMS는 아주 중요한 위치를 차지하고 있으며 그중에 서도 ORACLE은 안정성과 많은 기능으로 대기업에서 많이 사용되 는 DMBS이다. 가능하다면 SQL학습은 ORACLE로 하는것이 좋다고 판단 할수있다.
